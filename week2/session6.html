#### Named Slots and Fallback Content It's often useful to have multiple slots in our reusable components We can achieve this through [Named Slots](https://vuejs.org/guide/components/slots.html#named-slots) ``` <div class="container"> <header> <slot name="header"></slot> </header> <main> <slot> <!-- fallback content here --> </slot> </main> <footer> <slot name="footer"></slot> </footer> </div> ```
How to make an Alert Box for warning, ok messages?
#### Multiple models, single component Complex components, such as forms or controls, need to work with many reactive variables Vue offers this syntax for [multiple variable binding](https://vuejs.org/guide/components/events.html#usage-with-v-model): ``` <MyComponent v-model:variable-name="aReactiveValue" v-model:other-variable="otherReactiveValue" ... /> ```
